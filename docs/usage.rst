Installation
------------

⚠️ Currently, the best way is to install from this repository. This will
change soon.

Clone the repository
^^^^^^^^^^^^^^^^^^^^

.. code:: bash

   git clone git@github.com:Wicwik/PEFT-Factory.git

Build the wheel
^^^^^^^^^^^^^^^

.. code:: bash

   make build

Install wiht pip
^^^^^^^^^^^^^^^^

.. code:: bash

   pip install dist/peftfactory-0.9.4.dev0-py3-none-any.whl
