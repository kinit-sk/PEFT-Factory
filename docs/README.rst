   PEFT-Factory is a fork of
   `LLaMa-Factory <https://github.com/hiyouga/LLaMA-Factory>`__ ❤️,
   upgraded with easy to use **PEFT interface**, support for
   **HuggingFace PEFT methods** and **datasets** for benchmarking PEFT.
